## 🚀 Sobre o projeto

Este projeto foi desenvolvido para apresentar minha trajetória, habilidades técnicas e principais projetos na área de desenvolvimento de software.

Após anos no esporte de alto rendimento, iniciei minha transição de carreira para a tecnologia, levando comigo valores como disciplina, resiliência, trabalho em equipe e aprendizado contínuo.

O objetivo deste portfólio é reunir meus projetos e demonstrar minha evolução como desenvolvedor.

## ⚙️ Executando o projeto

Clone o repositório

```bash
git clone https://github.com/Joda33/portifoli.git
```

Entre na pasta

```bash
cd portifoli
```

Instale as dependências

```bash
npm install
```

Execute o projeto

```bash
npm run dev
```

Abra

```
http://localhost:5173
```
